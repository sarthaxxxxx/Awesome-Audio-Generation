## How to Contribute

We welcome contributions from the community! If you find a new paper or resource that should be included, please follow these steps:

1.  **Fork** the repository.
2.  **Create a new branch** for your addition.
3.  **Add the paper** to the appropriate category in the following format:
    ```markdown
    - [Paper Title](link_to_paper) - Authors. **Conference/Journal Year**. [[Code](link_to_code)]
    ```
4.  **Submit a Pull Request** with a clear description of the addition.

Let's build the most comprehensive resource for audio generation together!
